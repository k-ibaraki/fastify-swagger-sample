# fastify-swagger-sample

fastify-swaggerでOpenAPIのドキュメントをコードから自動生成してみた

## install

```sh
npm i
```

## 実行

```sh
npm run dev
```

- サーバー起動後、自動生成されたOpenAPIドキュメントにアクセス
  - WEB-UI : <http://127.0.0.1:3000/docs/>
  - JSON : <http://127.0.0.1:3000/docs/json>
  - YAML : <http://127.0.0.1:3000/docs/yaml>
